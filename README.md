# RD4 Assembler

This is an assembler made for the Red Diamond 4 redstone computer.

Export format: `.mcstructure`

## How to use

Either [Github Pages](solarflurry.github.io/RD4-assembler) or building it yourself:

```sh
git clone https://github.com/SolarFlurry/RD4-assembler.git
cd RD4-assembler
npm install
npm run prepare
npm run build:dev
npm run start
```
